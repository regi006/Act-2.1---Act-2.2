## Implementación de un ADT de Double Linked List

Complementa el programa de la Actividad en clase 10: Listas doblemente ligadas y agrega las siguientes opciones:
* Ordena la lista usando quicksort.
* Invierte la lista sin usar memoria adicional (in place).
* Constructor de copia (deep copy constructor) que crea una lista nueva con base en otra lista recibida como parámetro.
* Sobrecarga del operador de asignación que permite copiar el contenido de una una lista (lado izquierdo de la asignación) en otra lista (lado derecho de la asignación). e.g., lista2 = lista1, copiará lista1 en lista2.
